Documentação API Terminal TSE

POST /finalizarVotacaoAtual

Obtém o status da votação atual na urna (finalizado)
Parâmetro (String status, String ipUrna)

GET /confirmarVotacaoAtual

Verifica se a votação atual já foi finalizada
Parâmetro (String ipTerminal)
Retorna como parâmetro um boolean (true ou false)

POST /tempoParaUrna

Obtém o id do voto atual da urna (codigo da Urna)
Parâmetro (Long codUrna, ipTerminal)

GET /addTempo

Verifica se a urna solicitou tempo
Parâmetro (String ipUrna)
Retorna como parâmetro um boolean (true ou false)
